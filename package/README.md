# snet-token-contracts
Includes token contracts, migrations, tests

### SingularityNetToken
* ERC-20 implementation for SingularityNET AGIX/Fetch(ASI) Tokens

## Deployed Contracts

## Requirements
* [Node.js](https://github.com/nodejs/node) (12+)
* [Npm](https://www.npmjs.com/package/npm)

## Install

### Dependencies
```bash
npm install
```

### Test 
```bash
npm run test
```

## Package
```bash
npm run package-npm
```
